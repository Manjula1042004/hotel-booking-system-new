# 🏨 Hotel Booking System

A comprehensive hotel booking system built with Spring Boot, Thymeleaf, and MySQL.

## 🚀 Live Demo
**Application URL**: [Will be updated after deployment]

## 📋 Features
- User Registration & Authentication
- Hotel & Room Management
- Booking System with Date Conflict Check
- RazorPay Payment Integration
- Admin Dashboard
- Search & Filter Hotels

## 🛠️ Tech Stack
- **Backend**: Spring Boot, JPA, Spring Security
- **Frontend**: Thymeleaf, Bootstrap, CSS
- **Database**: MySQL
- **Payment**: RazorPay Integration
- **Deployment**: Docker, Render

## 🔑 Default Credentials
- **Admin**: admin@hotel.com / admin123
- **Users**: Register new account via registration page
